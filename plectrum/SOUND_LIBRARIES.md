# Metronome Sound Libraries

Here are some great places to find metronome sounds:

## Free Sound Libraries:

1. **Freesound.org** - https://freesound.org
   - Search for "metronome" or "click"
   - Free with attribution
   - Many high-quality metronome sounds

2. **Zapsplat** - https://www.zapsplat.com
   - Free with account
   - Search "metronome"
   - Professional quality sounds

3. **Pixabay** - https://pixabay.com/music/search/metronome/
   - Completely free, no attribution needed
   - Search "metronome"

4. **Adobe Stock** - https://stock.adobe.com
   - Some free sounds available
   - Search "metronome click"

5. **BBC Sound Effects Library** - https://sound-effects.bbcrewind.co.uk
   - Free for personal use
   - Search "metronome"

## How to Use Sound Files:

1. Download a metronome sound file (WAV or MP3 format)
2. Place it in `assets/sounds/` directory:
   - `metronome-accent.wav` (for beat 1)
   - `metronome-regular.wav` (for other beats)
3. Update `src/utils/metronome.ts` to use these files instead of generated sounds

## Recommended Search Terms:
- "metronome click"
- "metronome tick"
- "metronome beat"
- "click track"
- "drum machine click"

## File Format:
- **WAV** format is recommended (uncompressed, better quality)
- **MP3** also works (smaller file size)
- Keep files short (under 0.5 seconds)
- Mono or stereo both work
