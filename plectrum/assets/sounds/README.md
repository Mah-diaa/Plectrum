# Metronome Sound Files

To use your own metronome sounds:

1. **Download metronome sounds** from one of these free libraries:
   - [Freesound.org](https://freesound.org) - Search "metronome"
   - [Zapsplat](https://www.zapsplat.com) - Search "metronome"
   - [Pixabay](https://pixabay.com/music/search/metronome/)
   - [BBC Sound Effects](https://sound-effects.bbcrewind.co.uk)

2. **Place the files here**:
   - `metronome-accent.wav` - For the accent beat (beat 1 of every 4)
   - `metronome-regular.wav` - For regular beats

3. **File requirements**:
   - Format: WAV or MP3
   - Duration: Short (under 0.5 seconds recommended)
   - Quality: Any quality works, but shorter files load faster

4. **The app will automatically use these files** if they exist, otherwise it will use generated sounds.

## Recommended Search Terms:
- "metronome click"
- "metronome tick"  
- "click track"
- "drum machine click"
